
## 第一部分：三角函数与反三角函数体系

### 1. 六种基本函数之间的恒等关系

三角函数之间的所有关系都源于定义和勾股定理。

#### (1) 倒数与商数关系

- 倒数关系:
    
    $$\tan x = \frac{1}{\cot x}, \quad \sec x = \frac{1}{\cos x}, \quad \csc x = \frac{1}{\sin x}$$
    
- 商数关系:
    
    $$\tan x = \frac{\sin x}{\cos x}, \quad \cot x = \frac{\cos x}{\sin x}$$
    

#### (2) 毕达哥拉斯恒等式 (平方关系)

这是最基础的关系，其他两个由其推导而来。

- $$\sin^2 x + \cos^2 x = 1$$
    
- $$\tan^2 x + 1 = \sec^2 x$$
    
- $$\cot^2 x + 1 = \csc^2 x$$
    

### 2. 高阶转换公式

#### (1) 倍角与降幂公式

这些公式在积分中用于**降阶**，非常重要。

- 二倍角公式（余弦）:
    
    $$\cos 2x = \cos^2 x - \sin^2 x = 2\cos^2 x - 1 = 1 - 2\sin^2 x$$
    
- 降幂公式（积分利器）:
    
    $$\sin^2 x = \frac{1 - \cos 2x}{2}, \quad \cos^2 x = \frac{1 + \cos 2x}{2}$$
    

#### (2) 辅助角公式

用于将 $\sin x$ 和 $\cos x$ 的线性组合转化为单一的正弦函数：

$$a \sin x + b \cos x = \sqrt{a^2 + b^2} \sin(x + \phi)$$

其中，$\phi$ 满足 $\cos \phi = \frac{a}{\sqrt{a^2 + b^2}}$ 且 $\sin \phi = \frac{b}{\sqrt{a^2 + b^2}}$。

#### (3) 万能公式 (Universal Substitution Formulas)

令 $t = \tan(x/2)$，可将所有三角函数代数化，常用于化简复杂的有理积分：

$$\sin x = \frac{2t}{1 + t^2}, \quad \cos x = \frac{1 - t^2}{1 + t^2}, \quad dx = \frac{2}{1 + t^2} dt$$

### 3. 反三角函数关系

#### (1) 互余关系

$$\arcsin x + \arccos x = \frac{\pi}{2}, \quad \arctan x + \text{arccot } x = \frac{\pi}{2}$$

#### (2) 复合与等价代换

- **倒数关系:** $\text{arcsec } x = \arccos (1/x)$, $\text{arccsc } x = \arcsin (1/x)$
    
- **复合函数:** $\sin(\arccos x) = \sqrt{1 - x^2}, \quad \tan(\arcsin x) = \frac{x}{\sqrt{1 - x^2}}$
    
- 反正切和差:
    
    $$\arctan x + \arctan y = \arctan \left(\frac{x+y}{1-xy}\right) + k\pi$$

## 第二部分：微分和不定积分速查表

### 1. 导数与微分表 (Differentiation Formulas)

#### (1) 三角函数的导数

- $$(\sin x)' = \cos x$$
    
- $$(\cos x)' = -\sin x$$
    
- $$(\tan x)' = \sec^2 x$$
    
- $$(\cot x)' = -\csc^2 x$$
    
- $$(\sec x)' = \sec x \tan x$$
    
- $$(\csc x)' = -\csc x \cot x$$
    

#### (2) 反三角函数的导数

- $$(\arcsin x)' = \frac{1}{\sqrt{1 - x^2}}$$
    
- $$(\arccos x)' = -\frac{1}{\sqrt{1 - x^2}}$$
    
- $$(\arctan x)' = \frac{1}{1 + x^2}$$
    
- $$(\text{arccot } x)' = -\frac{1}{1 + x^2}$$
    
- $$(\text{arcsec } x)' = \frac{1}{|x|\sqrt{x^2 - 1}}$$
    
- $$(\text{arccsc } x)' = -\frac{1}{|x|\sqrt{x^2 - 1}}$$
    

### 2. 不定积分表 (Indefinite Integration Formulas)

#### (1) 三角函数的积分

- $$\int \sin x dx = -\cos x + C$$
    
- $$\int \cos x dx = \sin x + C$$
    
- $$\int \tan x dx = -\ln|\cos x| + C$$
    
- $$\int \cot x dx = \ln|\sin x| + C$$
    
- $$\int \sec x dx = \ln|\sec x + \tan x| + C$$
    
- $$\int \csc x dx = \ln|\csc x - \cot x| + C$$
    
- $$\int \sec^2 x dx = \tan x + C$$
    
- $$\int \csc^2 x dx = -\cot x + C$$
    

#### (2) 反三角函数相关（标准积分形式）

- $$\int \frac{1}{\sqrt{a^2 - x^2}} dx = \arcsin \left(\frac{x}{a}\right) + C$$
    
- $$\int \frac{1}{a^2 + x^2} dx = \frac{1}{a} \arctan \left(\frac{x}{a}\right) + C$$
    
- $$\int \frac{1}{\sqrt{x^2 - a^2}} dx = \ln|x + \sqrt{x^2 - a^2}| + C$$
    
- $$\int \frac{1}{\sqrt{x^2 + a^2}} dx = \ln|x + \sqrt{x^2 + a^2}| + C$$
    
- $$\int \frac{1}{x\sqrt{x^2 - a^2}} dx = \frac{1}{a} \text{arcsec} \left(\frac{|x|}{a}\right) + C$$

## 第三部分：积分方法论与定积分技巧

### 1. 不定积分方法

- **凑微分 (第一类换元):** 识别 $f(\varphi(x)) \varphi'(x) dx$，令 $u = \varphi(x)$。
    
- **分部积分:** $\int u dv = uv - \int v du$ (遵循 LIATE 选 $u$ 原则)。
    
- **三角代换 (第二类换元):** 含有 $\sqrt{a^2 \pm x^2}$ 等根式时，用三角函数代换以消去根号。
    
- **有理函数:** 对 $P(x)/Q(x)$ 进行**部分分式分解**。
    

### 2. 定积分技巧与公式

- 牛顿-莱布尼茨公式:
    
    $$\int_a^b f(x) dx = F(b) - F(a)$$
    
- **换元原则:** $\int_a^b f(x) dx = \int_{\alpha}^{\beta} f[\varphi(t)] \varphi'(t) dt$，**上下限必须同步更换**。
    
- **奇偶性:** 当积分区间为 $[-a, a]$ 时：
    
    - $f(x)$ 为奇函数，则 $\int_{-a}^{a} f(x) dx = 0$。
        
    - $f(x)$ 为偶函数，则 $\int_{-a}^{a} f(x) dx = 2\int_{0}^{a} f(x) dx$。
        
- 变上限积分求导:
    
    $$\frac{d}{dx} \int_{h(x)}^{g(x)} f(t) dt = f(g(x)) \cdot g'(x) - f(h(x)) \cdot h'(x)$$
